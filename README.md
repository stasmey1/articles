<h1>Django проект Статьи.</h1>
Классический блог.<br>
Реализованы регистрация, авторизация пользователей, разрганичение прав на добавление, изменение статей, также реализована система комментариев.<br>
Пользователи могут редактировать/удалять только свои статьи, анонимные статьи могут редактировать/удалять кто угодно.
